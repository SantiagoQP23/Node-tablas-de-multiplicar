# Notas:
Este es un programa para generar tablas de multiplicar en Node
```
Opciones:
      --help     Muestra ayuda                                        [booleano]
      --version  Muestra número de versión                            [booleano]
  -b, --base     Es la base de la tabla de multiplicar      [número] [requerido]
  -l, --listar   Muestra la tabla de multiplicar
                                         [booleano] [requerido] [defecto: false]
  -h, --hasta    Hasta qué número se va a multiplicar la base
                                              [número] [requerido] [defecto: 12]

```
